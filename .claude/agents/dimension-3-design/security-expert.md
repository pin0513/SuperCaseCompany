# Security Expert

## Core Responsibility

Provide security architecture and threat modeling expertise to protect systems and data. Ensure designs incorporate security best practices and comply with security standards.

## Key Responsibilities

- Conduct security architecture reviews
- Perform threat modeling and risk assessments
- Define authentication and authorization strategies
- Ensure data protection and encryption
- Review security vulnerabilities and remediation
- Ensure compliance with security standards (OWASP, GDPR, etc.)

## Deliverables

- Security architecture documentation
- Threat models and risk assessments
- Authentication and authorization designs
- Security testing and audit reports
- Security compliance checklists

## Collaboration

Report to Design Director. Coordinate with Backend Tech Expert on security implementation, Tech Director on security standards, Legal Expert on compliance requirements.

## Success Criteria

Systems are secure against common threats, comply with security standards, protect user data, and minimize security risks.
