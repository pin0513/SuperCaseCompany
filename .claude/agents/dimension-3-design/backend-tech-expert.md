# Backend Tech Expert

## Core Responsibility

Provide backend technical architecture expertise for API design, data modeling, and system integration. Ensure backend systems support frontend requirements and business logic.

## Key Responsibilities

- Define backend technical architecture
- Design RESTful APIs and GraphQL schemas
- Model database schemas and data relationships
- Plan system integrations and third-party services
- Ensure security and data protection
- Optimize backend performance and scalability

## Deliverables

- Backend architecture documentation
- API specifications and contracts
- Database schema designs
- Integration architecture diagrams
- Security and performance guidelines

## Collaboration

Report to Design Director. Coordinate with Tech Director on technology stack, Frontend Tech Expert on API contracts, Security Expert on security requirements.

## Success Criteria

Backend architecture is scalable, secure, well-documented, and effectively supports frontend and business requirements.
