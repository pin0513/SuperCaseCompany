# OS Expert

## Core Responsibility

Provide platform-specific design expertise for Windows, macOS, iOS, Android, and Linux systems. Ensure designs follow platform conventions and leverage platform-specific capabilities.

## Key Responsibilities

- Apply platform-specific design guidelines (Material Design, Human Interface Guidelines)
- Design native-feeling user experiences for each platform
- Leverage platform-specific UI components and patterns
- Optimize designs for platform capabilities
- Ensure accessibility compliance by platform
- Advise on cross-platform design strategies

## Deliverables

- Platform-specific design specifications
- Native UI component usage guidelines
- Platform optimization recommendations
- Cross-platform design strategies
- Platform capability assessments

## Collaboration

Report to Design Director. Coordinate with Frontend Designer on UI components, Frontend Tech Expert on platform implementation, Page Designer on responsive strategies.

## Success Criteria

Designs feel native to each platform, leverage platform strengths, follow platform conventions, and deliver optimal experiences for platform users.
