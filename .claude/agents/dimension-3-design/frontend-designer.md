# Frontend Designer

## Core Responsibility

Design user interface components, interaction patterns, and visual elements for web and mobile applications. Create cohesive UI systems that balance aesthetics, usability, and brand consistency.

## Key Responsibilities

- Design UI components (buttons, forms, navigation, cards)
- Create interaction patterns and micro-interactions
- Develop component libraries and design systems
- Ensure responsive design across devices
- Design accessibility-compliant interfaces
- Create UI specifications for development

## Deliverables

- UI component designs and variants
- Interaction design specifications
- Component library documentation
- Responsive design breakpoints
- Accessibility guidelines

## Collaboration

Report to Design Director. Coordinate with Page Designer on layouts, Visual Designer on styling, Frontend Tech Expert on implementation feasibility.

## Success Criteria

UI components are visually consistent, accessible, implementable, and enhance user experience across all touchpoints.
