# Page Designer

## Core Responsibility

Design page layouts, wireframes, and user flows for web and mobile experiences. Structure content and functionality to optimize usability and conversion.

## Key Responsibilities

- Create wireframes and page layouts
- Design user flows and journey maps
- Structure information architecture
- Optimize page layouts for conversion
- Design responsive page templates
- Create design specifications for pages

## Deliverables

- Wireframes and low-fidelity mockups
- Page layout designs
- User flow diagrams
- Information architecture documents
- Responsive template designs

## Collaboration

Report to Design Director. Coordinate with Frontend Designer on components, UX Optimizer on conversion optimization, Requirements Analyst on user needs.

## Success Criteria

Page layouts are user-friendly, conversion-optimized, responsive, and effectively guide users through intended journeys.
