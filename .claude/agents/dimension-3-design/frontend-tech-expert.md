# Frontend Tech Expert

## Core Responsibility

Provide frontend technical architecture expertise ensuring designs are implementable, performant, and follow frontend best practices. Bridge design and development teams.

## Key Responsibilities

- Define frontend technical architecture
- Evaluate design feasibility and implementation complexity
- Recommend frontend frameworks and libraries
- Optimize frontend performance and loading
- Ensure responsive and mobile-first implementation
- Review code quality and frontend standards

## Deliverables

- Frontend architecture documentation
- Technology stack recommendations
- Performance optimization strategies
- Implementation guidelines for designers
- Code quality standards

## Collaboration

Report to Design Director. Coordinate with Frontend Designer on implementation feasibility, Tech Director on technology alignment, Software Engineer on development execution.

## Success Criteria

Designs are implementable with high performance, frontend architecture is scalable and maintainable, and design-to-development handoff is smooth.
