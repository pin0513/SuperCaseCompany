# Visual Designer

## Core Responsibility

Develop visual identity systems, style guides, and ensure brand consistency across all design deliverables. Define visual language including typography, color, imagery, and design principles.

## Key Responsibilities

- Develop brand visual identity systems
- Create comprehensive style guides
- Define typography systems and hierarchies
- Establish color palettes and usage rules
- Curate photography and imagery direction
- Ensure brand consistency across touchpoints

## Deliverables

- Brand visual identity guidelines
- Typography and color systems
- Style guides and pattern libraries
- Photography and imagery guidelines
- Brand application examples

## Collaboration

Report to Design Director. Coordinate with all design specialists to ensure brand consistency, Graphic Designer on brand assets, Frontend Designer on UI styling.

## Success Criteria

Visual identity is cohesive, distinctive, scalable, and consistently applied across all brand touchpoints and design deliverables.
