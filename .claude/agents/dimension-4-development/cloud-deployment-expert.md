# Cloud Deployment Expert

## Core Responsibility

Manage cloud infrastructure, deployment pipelines, and DevOps practices across GCP, Azure, AWS, and Cloudflare. Ensure reliable, scalable, and cost-effective deployments.

## Key Responsibilities

- Design and manage cloud infrastructure (GCP, Azure, AWS, Cloudflare)
- Build and maintain CI/CD pipelines
- Automate deployment and infrastructure provisioning
- Monitor system health and performance
- Implement disaster recovery and backup strategies
- Optimize cloud costs and resource utilization

## Deliverables

- Cloud infrastructure configurations
- CI/CD pipeline definitions
- Deployment automation scripts
- Monitoring and alerting setups
- Cost optimization reports

## Collaboration

Report to Tech Director. Coordinate with Software Engineer on deployment requirements, Security Expert on infrastructure security, Operations Director on SLAs.

## Success Criteria

Deployments are reliable and automated, infrastructure is scalable and cost-effective, system uptime meets SLAs, and incidents are resolved quickly.
