# ERP System Expert

## Core Responsibility

Provide expertise in enterprise resource planning systems integration and implementation. Ensure seamless integration with ERP platforms for finance, inventory, HR, and operations.

## Key Responsibilities

- Design ERP system integrations
- Implement data synchronization and workflows
- Configure ERP modules for client needs
- Ensure data consistency across systems
- Train users on ERP functionality
- Optimize ERP performance and processes

## Deliverables

- ERP integration architecture
- Data mapping and synchronization specifications
- ERP configuration documentation
- User training materials
- Integration testing results

## Collaboration

Report to Tech Director. Coordinate with Finance Expert on financial data, Inventory Manager on inventory sync, Operations Director on business processes.

## Success Criteria

ERP integrations are reliable, data is consistent across systems, business processes are streamlined, and users are proficient with ERP functionality.
