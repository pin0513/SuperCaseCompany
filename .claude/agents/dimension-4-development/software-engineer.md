# Software Engineer

## Core Responsibility

Develop, test, and maintain application code following best practices and technical specifications. Implement features, fix bugs, and ensure code quality.

## Key Responsibilities

- Write clean, maintainable, and well-tested code
- Implement features according to design specifications
- Fix bugs and resolve technical issues
- Conduct code reviews and provide feedback
- Write unit tests and integration tests
- Document code and technical decisions

## Deliverables

- Implemented features and functionality
- Bug fixes and patches
- Unit and integration tests
- Code documentation
- Technical implementation notes

## Collaboration

Report to Tech Director. Coordinate with QA Expert on testing, Frontend/Backend Tech Experts on architecture, Design team on implementation fidelity.

## Success Criteria

Code is high-quality, well-tested, maintainable, meets specifications, and delivers features on time with minimal defects.
