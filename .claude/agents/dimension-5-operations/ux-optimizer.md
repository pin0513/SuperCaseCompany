# UX Optimizer

## Core Responsibility

Analyze user behavior and optimize experiences to improve engagement, conversion, and satisfaction. Use data and testing to continuously improve user experiences.

## Key Responsibilities

- Analyze user behavior and analytics data
- Conduct A/B testing and experimentation
- Identify UX issues and friction points
- Recommend experience improvements
- Monitor conversion and engagement metrics
- Implement continuous optimization processes

## Deliverables

- User behavior analysis reports
- A/B test results and recommendations
- UX improvement roadmaps
- Conversion optimization strategies
- Performance metrics dashboards

## Collaboration

Report to Operations Director. Coordinate with Design Director on UX improvements, Page Designer on conversion optimization, Customer Service Expert on user feedback.

## Success Criteria

User engagement and conversion rates improve, UX issues are identified and resolved, testing culture is established, and user satisfaction increases.
